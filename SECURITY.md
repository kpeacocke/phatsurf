
# Security Policy

## Reporting Security Issues

We take security vulnerabilities seriously and appreciate your efforts to report them responsibly. If you discover a security issue in this project, please follow the steps below:

1. **Contact Us**
   - Email: **krpeacocke@gmail.com**
   - Include "SECURITY ISSUE" in the subject line.

2. **Provide Details**
   - Describe the nature of the vulnerability.
   - Include steps to reproduce the issue, if possible.
   - Provide any additional information to help us assess and resolve the problem.

3. **Expect a Response**
   - We aim to acknowledge receipt of security issues within **48 hours**.
   - We will work with you to understand the issue and provide a timeline for a fix.

## Supported Versions

Only the latest released version of the app is supported for security updates. Older versions may not receive fixes.

## Reporting Non-Security Issues

For general bugs and feature requests, please use the [Issues](https://github.com/yourusername/repo-name/issues) tab.

Thank you for helping us maintain the security and integrity of the Surfboard Recommender and Alerts App!
